# Build Week Submission Checklist

- [ ] Public demo URL is stable.
- [ ] Deterministic broken → fixed flow completes in under 90 seconds.
- [ ] GPT-5.6 mode is tested with the configured account model.
- [ ] `npm run verify` passes.
- [ ] GitHub Actions release gate is green.
- [ ] Production dependency audit has no high or critical findings.
- [ ] Public YouTube demo is under 3 minutes and includes voiceover.
- [ ] Demo explains what was built, how Codex was used, and how GPT-5.6 was used.
- [ ] Repository link is included.
- [ ] README contains setup and judge instructions.
- [ ] MIT license is present.
- [ ] `/feedback` session ID is recorded in the Devpost form.
- [ ] Submission uses the Work and Productivity track.
- [ ] No production tenant data, credentials, internal URLs, or private customer information are present.
